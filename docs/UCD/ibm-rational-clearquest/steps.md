# Process Steps

## Add Related Change Request

Add a Related Change Request link to a record. (Note: This will remove all other links.)

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| Hostname | textBox | The IBM Rational ClearQuest's Hostname. Example: quagmire.rtp.raleigh.ibm.com | Yes |
| Repository | textBox | The IBM Rational ClearQuest's repository containing the record. Example: DOC | Yes |
| Database | textBox | The IBM Rational ClearQuest's database containing the record. Example: SAMPL | Yes |
| Database ID | textBox | The 8 digit Rational ClearQuest database identifier that uniquely identifies an object. | Yes |
| Record ID | textBox | The 8 digit Rational ClearQuest database identifier that uniquely identifies the record object. | Yes |
| Related Change Request Title | textBox | The title of the Related Change Request. | No |
| Related Change Request Link | textBox | The link of the Related Change Request. | Yes |
| Username | textBox | The username to use to authenticate with Rational Clearquest. | Yes |
| Password | secureBox | The password to use to authenticate with Rational Clearquest. | Yes |
| Accept All Certificates | checkBox | Select to accept all self signed certifcates. | No |
