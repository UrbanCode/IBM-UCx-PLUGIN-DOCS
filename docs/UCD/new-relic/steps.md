# New Relic - Process Steps

## Deployment Notification

Send a deployment notifications to New Relic.

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| New Relic URL | textBox | New Relic Deployment Notification URL | Yes |
| New Relic API key | textBox | New Relic API key. | Yes |
| Application Name | textBox | The name of the application deployed. | Yes |
| Description | textBox | The details of the application deployed. | Yes |
| User | textBox | The Application which performed the deployment. | Yes |
| Version | textBox | The version of the application deployed. | Yes |

## Upload DSYM File

Uploads Xcode DSYM file to NewRelic server for Desymbolication.

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| Application ID | textBox | New Relic mobile application id | Yes |
| Product Name | textBox | Application Product Name. | Yes |
| XCArchive path | textBox | File path to XCArchive | Yes |
