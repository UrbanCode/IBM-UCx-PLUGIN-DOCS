# Process Steps

## Deploy IPA

Deploy IPA to AirWatch instance

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| AirWatch URL | textBox | AirWatch API base URL. | Yes |
| User Name | textBox | AirWatch username | Yes |
| Password | secureBox | AirWatch password | Yes |
| IPA Filepath | textBox | IPA file path | Yes |
| Application Name | textBox | Application name | Yes |
| Auth Token | secureBox | AirWatch authentication token | Yes |
| Supported Devices | textBox | Supported devices for this application (tablet, phone or both) | No |

## Assign IPA to group

Deploy IPA to AirWatch instance

| Name | Type | Description | Required |
| ---- | ---- | ----------- | -------- |
| AirWatch URL | textBox | AirWatch API base URL. | Yes |
| User Name | textBox | AirWatch username | Yes |
| Password | secureBox | AirWatch password | Yes |
| Application Name | textBox | Application name | Yes |
| Auth Token | secureBox | AirWatch authentication token | Yes |
| Smart Group | textBox | AirWatch smart group to assign IPA to | Yes |
