## Wait

Use this step to tell the process to wait for the specified number of seconds before the process resumes.


| Name | Type | Description                                                                                                          | Required |
| ---- | ---- | -------------------------------------------------------------------------------------------------------------------- | -------- |
| Duration (seconds) | String | The number of seconds to wait. | Yes |
