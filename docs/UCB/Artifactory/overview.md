# Overview


### Overview




### Compatibility

The IBM DevOps Build automation plug-in for Artifactory works with Artifactory version 4. The Artifactory Pro edition is also supported. This plug-in requires version 6.1.2 or later of IBM DevOps Build. This plug-in is supported to run on all operating systems that are supported by the IBM DevOps Build agent.

### Installation

No special steps are required for installation.
